## Quick Start

1. **Install the extension**
2. **Open a Python file**
3. **Try manual conversion:**
   - Open Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`)
   - Type "VL: Convert Current File to VL"
   - See the token-optimized version!

## Commands

- `VL: Convert Current File to VL` - Convert Python/JS to VL format
- `VL: Compile VL to Target Language` - Compile VL back to Python/JS/TS
- `VL: Show Cost Savings Dashboard` - View analytics and features
- `VL: Toggle Transparent Mode` - Enable/disable auto-optimization
- `VL: Reset Statistics` - Clear token savings statistics
